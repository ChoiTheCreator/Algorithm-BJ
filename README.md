# コーディングテストについて

## 使用言語
コーディングテストは **JavaScript** と **Java** で行います。

しかし、おそらく JavaScript を使用する可能性が高いです。

## 環境設定
- **JavaScript**: Node.js (推奨バージョン: 18以上)
- **Java**: JDK 17 以上

## 実行方法
### JavaScript
```sh
node test.js
```

### Java
```sh
javac Test.java
java Test
```

## 問題のフォーマット
各問題は `problems/` フォルダに格納されています。

## 注意事項
- コードの最適化を意識してください。
- 可読性の高いコードを書くようにしましょう。
- 問題ごとに適切なテストケースを作成してください。
